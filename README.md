## 🇬🇧 English

Welcome to the **ji2c** repository by Pibiger Technology. This repository is dedicated to the **JI2C Module**, a specialized USB-to-I2C adapter designed for interacting with I2C devices and programming EEPROMs.

### 🌟 Overview

The Pibiger JI2C Module provides a straightforward and reliable way to connect I2C sensors, displays, and memory chips directly to your computer via USB. It is an essential tool for hardware debugging, prototyping, and production programming.

### 📦 Repository Contents

This repository contains the necessary software and documentation for the JI2C Module:

- **JI2C For I2C.pdf**: User manual focusing on general I2C device communication.
- **JI2C(FOR EEPROM)_EN.pdf**: English user manual specifically for EEPROM reading and writing operations.
- **JI2C(For-I2C-Devices).zip**: The Windows software package for general I2C device interaction.
- **JI2C(For-EEPROM).zip**: The Windows software package tailored for EEPROM programming.

### 🚀 Getting Started

1. **Hardware Connection**: Connect the JI2C Module to your PC via USB. Connect the I2C pins (SDA, SCL, GND, VCC) to your target device.
2. **Download Software**: Depending on your use case, download either `JI2C(For-I2C-Devices).zip` or `JI2C(For-EEPROM).zip`.
3. **Read the Manual**: Review the corresponding PDF manual to understand the software interface and configuration options.
4. **Run the Application**: Extract the zip file and run the executable to start communicating with your I2C devices.
